# AI Software Engineering Assistant (VOID)

A premium AI-powered engineering platform that unifies debugging, code analysis, optimization, and testing.

## Project Structure

This project is organized into two main parts:

- **[/frontend](file:///c:/Users/USER/OneDrive/Desktop/new%20one/frontend)**: A React + Vite application with a Ghibli/Nothing-inspired monochrome aesthetic.
- **[/backend](file:///c:/Users/USER/OneDrive/Desktop/new%20one/backend)**: A Node.js + Express server for handling AI requests (OpenRouter API).

## Getting Started

### Frontend
1. Navigate to `/frontend`
2. Run `npm install`
3. Run `npm run dev`

### Backend
1. Navigate to `/backend`
2. Run `npm install`
3. Configure `.env` with your API keys.
4. Run `npm start` (or `npm run dev`)
